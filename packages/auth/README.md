# @repro-v2/auth

Better Auth server factory and React client for frontends.

| Export | Purpose |
| --- | --- |
| `@repro-v2/auth` | `createAuth(db)` — server-side Better Auth instance |
| `@repro-v2/auth/react` | `createAuthReactClient(baseUrl)` — React hooks client |

Console example: `apps/console/src/lib/auth-client.ts`.
