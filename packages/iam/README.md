# @repro-v2/iam

Better Auth server factory (`createIam`) and React client for frontends.

| Export | Purpose |
| --- | --- |
| `@repro-v2/iam` | `createIam(db)` — server-side Better Auth with IAM feature plugins |
| `@repro-v2/iam/react` | `createIamReactClient(baseUrl)` — React hooks client |

Console example: `apps/console/src/lib/iam-client.ts`.
