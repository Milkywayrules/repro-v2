# @repro-v2/api-types

Shared API contract types and HTTP constants for clients and server.

| Export | Purpose |
| --- | --- |
| `@repro-v2/api-types` | Re-exports contract types |
| `@repro-v2/api-types/contract` | Envelopes, pagination meta, error shapes |
| `@repro-v2/api-types/constants` | `httpStatus`, `errorCodes`, `errorMessages`, `api.VERSION` |
| `@repro-v2/api-types/generated/openapi.json` | Checked-in OpenAPI spec (regenerate via root `bun run generate:openapi`) |

Monorepo type ownership: root [AGENTS.md](../../AGENTS.md).
